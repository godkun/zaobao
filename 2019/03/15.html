    # 2019.03.15

    [文章] package.json 参数详解，并且提供了该参数相应的案例：https://github.com/stereobooster/package.json

    [文章] 深入理解 TypeScript 中文版，在线阅读：https://github.com/jkchao/typescript-book-chinese

    如今社区已经存在部分翻译，但都似乎已经停止更新。于是在某天的某个冲动之下，这个 RP 就诞生了。

    [工具] Chrome 插件英雄榜，为优秀的 Chrome 插件写一本中文说明书，在线阅读：https://github.com/zhaoolee/ChromeAppHeroes

    里面有个一键分析网站技术栈的工具，不错。

    [类库] type-fest 是一个 TypeScript 扩充类型集合，如：JSONObject、JSONArray、Primitive：https://github.com/sindresorhus/type-fest
